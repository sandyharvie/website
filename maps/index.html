Maps

Maps is a web mapping service with real-time traffic updates, similar to Google Maps. Written from scratch in Java, it uses a k-d Tree for nearest neighbor searches, A* for routing, and autocorrect to suggest locations. Extensive caching is utilized on the frontend to allow for smooth panning and zooming, and on the backend to allow for quick routing.

Users are able to click on the map or enter cross-streets into an input field with autocorrect capabilities when specifying their start and end locations. Traffic is displayed and accounted for in real-time.
