Blast

Blast provides a scalable, robust platform that allows organizations to easily conduct inbound and outbound campaigns for marketing, informational, or other purposes.

For outbound communication, Blast leverages the Twilio API. Outbound campaigns allow organizations to schedule text messages and phones calls from a number of their choosing to a specified list of contacts. These contacts may be imported manually or from CSV files, and can be filtered by age, income, and location. If any of this information is not provided upon import, it is retrieved via the Whitepages API.

Blast also uses the Twilio API to support inbound communication. Inbound campaigns allows organizations to receive text messages and send a response based on keyword(s) in the message. This response may be static, or it may be generated via a WebHook specified by the organization during the creation of the campaign. Upon receiving a message as part of an inbound campaign, Blast retrieves basic information about the sender via the Whitepages API and adds them to a contact list. This allows for visual and statistical reporting of inbound communication and also makes it easy for the organization to remarket to these individuals.

Blast also provides an extensive API that allows for it to be easily integrated into other systems.
